# 🛒 Fullstack Challenge - E-commerce

Este es un proyecto de aplicación web tipo E-commerce desarrollado con **React.js**, aplicando la metodología **Atomic Design** y mejores prácticas de desarrollo frontend.

## 🚀 Presentación del Proyecto

### 📋 Información General
- **Nombre del Proyecto**: Fullstack Challenge - E-commerce
- **Tecnologías**: React, Vite, Tailwind CSS, Zustand, Axios.
- **API**: [FakeStore API](https://fakestoreapi.com/) para datos de productos en tiempo real.

### 🧠 Funcionalidades Core
- ✅ **Registro y Login**: Manejo de sesión persistente con `localStorage`.
- ✅ **Galería de Productos**: Carga dinámica desde la API real.
- ✅ **Buscador en Tiempo Real**: Filtra productos por nombre instantáneamente.
- ✅ **Paginación**: Navegación fluida entre grupos de productos.
- ✅ **Carrito de Compras**: Agregar, eliminar y actualizar cantidades con cálculo de totales.
- ✅ **Checkout**: Resumen de compra detallado y simulación de pago.

---

## 🧩 Arquitectura (Atomic Design)

El proyecto está organizado siguiendo el estándar de **Atomic Design**:
- **Atoms**: Componentes básicos (botones, inputs, badges).
- **Molecules**: Componentes compuestos (ProductCard, SearchBar).
- **Organisms**: Componentes complejos (Header, Footer, ProductGrid).
- **Templates**: Layouts estructurales (MainLayout).
- **Pages**: Vistas principales de la aplicación.

---

## 🛠️ Tecnologías y Herramientas
- **React.js**: Librería principal.
- **Vite**: Entorno de desarrollo ultrarrápido.
- **Tailwind CSS**: Estilos modernos y responsivos.
- **Zustand**: Gestión de estado simple y eficiente.
- **React Router**: Navegación SPA.
- **Axios**: Cliente HTTP para consumo de API.

## 🔧 Instalación y Uso

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/xaca/reto_fullstack
   ```
2. **Instalar dependencias**:
   ```bash
   npm install
   ```
3. **Ejecutar el proyecto**:
   ```bash
   npm run dev
   ```
